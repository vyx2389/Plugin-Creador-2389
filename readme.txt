Installation

Extract the archive file "plugin_maker.zip" Then uploaded into your web hosting or if work on the project in localhost simply copy it, Suggested in your WordPress folder. 
After you successfully copy or upload, for linux/mac don't forget change chmod 777 *.
you can use it open url address in browser, "http://localhost/yourwp/plugin-maker/" or "http://yourdomain/yourwp/plugin-maker/"
