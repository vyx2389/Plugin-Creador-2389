How it works? Eazy to Use

WordPress Plugin Maker can work with any web browser, either on your Local Network or on the Internet. Simply put side by side with WordPress CMS (copy-paste) when making plugins, then start a project then build it, done!

What it can do? With Code Builder

Through WordPress Plugin Maker you can develop WordPress Plugin and customized features very quickly. It can build plugins in an intuitive and fast way, like image size, custom post type or database, toolbox (metabox), admin bar (toolbar), shortcode and quicktags, plugin option, widget front-end/back-end, WP Ajax nopriv & auth, insert CSS, insert JS, Top level admin menu and other.

Experienced users? Not require high skill

You just need to make some css and javascript make your work look beautiful, if possible you can also edit the php code, it easily because it supports todo (IDE Editor), comments and variables have also been provided, simply use echo to see these variables.
